## 🛠️ Método de Inmersión por Asignatura
Cada archivo `.md` contiene secciones preconfiguradas para registrar las 7 fases del estudio conjunto:
1. **Reconocimiento del terreno:** Contextualización del problema.
2. **Identificación del texto-eje:** Elección justificada de la fuente primaria.
3. **Primera lectura:** Detección del movimiento estratégico del texto.
4. **Mapeo conceptual:** Estructura visual de las ideas.
5. **Segunda lectura:** Reconstrucción argumentativa formal.
6. **Confrontación dialéctica:** Intercambio de objeciones y defensas.
7. **Tercera lectura:** Elaboración del ensayo crítico final.

*¡Mucho éxito en el camino del pensamiento!*
